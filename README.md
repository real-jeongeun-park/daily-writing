# 델리쓰기 (Daily Writing)

**"글쓰기 습관의 시작"**

매일 짧게라도 글을 쓰는 습관을 길러주는 **데일리 글쓰기 앱**입니다. 짧은 숏폼 콘텐츠 소비로 인한 도파민 중독에서 벗어나, 하루 한 번 스스로의 생각을 글로 정리하는 건강한 습관을 만드는 것을 목표로 기획했습니다.

https://github.com/user-attachments/assets/f4c7f5df-484e-45dd-9cd5-cae2d5dea878

## 타겟 사용자

- 도파민 디톡스를 원하는 사람
- 글을 잘 쓰고 싶지만 매번 작심삼일로 끝나는 사람
- 글쓰기 능력 향상이 필요한 대학생

## 사용 기술 / 개발 환경

- **개발 툴**: App Inventor (앱 인벤터, 블록 기반 앱 개발 플랫폼)
- **데이터베이스**: TinyDB — 사용자 정보, 작성한 글 내용 및 목록 저장
- **AI 연동**: ChatBot 컴포넌트 — 글쓰기 소재 추천, 글쓰기 관련 질의응답 등 사용자의 글쓰기 과정을 보조
- **UI/UX 디자인**: Pixso — 정형화된 글쓰기 앱 디자인에서 벗어나 감각적이고 직관적인 UI로 설계
- **날짜 처리**: DatePicker 컴포넌트 — 날짜별 글쓰기 기능 구현

## 주요 기능

### 1. 회원 관리
- 회원가입 (닉네임 / 이메일 / 아이디 / 비밀번호)
- 로그인, 아이디 찾기, 비밀번호 찾기 (이메일 인증 기반)
- 환경설정을 통한 회원정보 수정 및 탈퇴 기능

### 2. 캘린더 기반 날짜별 글쓰기
- 캘린더에서 날짜를 선택해 해당 날짜에 쓴 글 열람 가능
- 오늘 날짜에 한해 새로운 글 작성 가능
- 이전에 작성한 글은 좌우 슬라이드로 넘겨보며 확인 가능

### 3. AI 도움받기
- 글쓰기가 막힐 때 AI 챗봇에게 소재 추천, 글쓰기 팁 등을 실시간으로 질문 가능
- 현재까지 작성 중인 글을 불러와 "어색한 부분 고치기" 등 맞춤형 도움을 받을 수 있도록 설계

### 4. 도움말
- 앱 사용법, 자주 묻는 질문(FAQ)을 슬라이드 형태로 제공

### 5. 환경설정
- 닉네임 / 비밀번호 변경
- 회원 탈퇴 (탈퇴 시 저장된 모든 데이터 삭제, 복구 불가 경고 제공)

## 서비스 구조

```
델리쓰기
├── 로그인
│   ├── 아이디 찾기
│   ├── 비밀번호 찾기
│   └── 회원가입
└── 메인
    ├── 캘린더 → 글쓰기 → AI 도움받기
    ├── 도움말
    └── 환경설정 → 탈퇴하기
```
